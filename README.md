# intercambiadore-de-calor-py-steamlit
Aplicación desarrollada en Python con Streamlit para el análisis de intercambiadores de calor.

## Características
- Cálculo de parámetros térmicos
- Interfaz interactiva
- Uso de NumPy para cálculos numéricos
- Resultados rápidos y precisos

## Tecnologías
- Python
- Streamlit
- NumPy

## Uso
Puedes ejecutar la aplicación localmente o acceder a la versión en línea (Streamlit Cloud).
